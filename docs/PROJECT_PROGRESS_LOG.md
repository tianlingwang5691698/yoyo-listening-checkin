## 2026-04-18 字体语言收敛

提交：

- `e9bf5f7 refactor: define minimal typography system`

变化：

- 新增并开始使用 6 个字体层级：
  - `type-brand`
  - `type-page-title`
  - `type-card-title`
  - `type-section-title`
  - `type-body`
  - `type-caption`
- 降低普通标题字重。

结论：

- 高级感不是靠粗字重和大字号堆出来的。
- 普通标题不应使用 `font-weight: 900`。
- `900` 应保留给成长页大数字或统计数字。

## 2026-04-18 课程页播放区收敛

相关提交：

- `2734785 refactor: simplify lesson playback surface`
- `a2e9140 refactor: remove lesson explainer panels`
- 待本次提交：课程页重复进度与播放控件收敛
- 待本次提交：课程页三遍节奏合并
- 待本次提交：课程页倍速显示修复

变化：

- 普通课程页不再展示 `音频来源`、`播放链路`、`播放状态：已就绪` 等技术链路。
- 删除文本来源、结果、历史等说明卡。
- 播放区和文本区更靠近主流程。
- 删除同屏重复的遍数信息。
- 播放键改为自绘控件，降低真机和调试器样式差异。
- 倍速只显示实际值，不做提醒说明。
- 独立三遍节奏卡被合并进播放器卡。
- 倍速切换使用 `×1.0` / `×0.9`。
- 倍速显示改为 JS 派生字段，避免真机不渲染 `toFixed()` 导致只显示 `×`。

结论：

- 课程页当前重点已从“隐藏调试信息”进入“单焦点播放体验”。
- 下一步应在真机确认播放键一致性，并继续评估是否合并顶部信息区和三遍节奏区。
- 当前需要真机确认：播放键、倍速按钮、三段状态是否符合预期。
- 倍速按钮应稳定显示 `×1.0` 或 `×0.9`。

提交：

- 待本次提交：音频页阶段入口与占位详情

变化：

- 音频页从统计/资源式结构改为级别与节目入口结构。
- 增加 A1-C2 级别选择，当前仅 A1 可用。

- Peppa / Unlock 1 / Songs 作为三条节目入口。

- 移除页面可见 Day，改为 A1 当前阶段。
- A1 阶段改为任务组合表达：基础输入组、复习加速组、综合冲刺组。
- 新增阶段详情页，展示轻提示、预计总时长和任务组合。
- 导航标题从 `A1 课程地图` 改为 `听力路径`。

- 音频页首页删除 Peppa / Unlock 1 / Songs 节目入口。
- 阶段命名改为 `听力组合 A/B/C`。
- 阶段二、阶段三详情页先显示 `暂时没有组合`。

- 音频页职责明确为“选择听什么”，不承载学习数据。
- A2-C2 当前只做前端占位，后续接内容池时再扩展数据结构。
- A1 不应叫课程地图；更适合用“听力路径”和“阶段组”表达。
- 阶段详情页承载组合细节，音频页保持入口职责。
- 音频页只保留入口层级，不重复展示阶段详情。
- 阶段二/三没有真实组合前，不应复用阶段一组合内容。
